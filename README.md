# CardVault

A vintage sports card knowledge base powered by Gas Town multi-agent system.

## Pipeline

1. **Ingest** - Photograph cards in a grid layout, upload, and auto-segment individual cards
2. **Identify** - AI agents identify each card (player, year, set, manufacturer, card number)
3. **Research** - Agents estimate condition, rarity, and market value
4. **Browse** - Web interface with filtering by name, price, rarity, sport, year, etc.
